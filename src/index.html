My Modern Web Tools

Sass

CSS Preprocessor

Probably the most popular used today. Sass gives you the ability to structure your CSS better, use variables, use functions and mixins. The bottom line is Sass is a powerful way to help you gain control of your CSS and makes writing CSS fun.

PostCSS

New Cool Kid on the block when working with CSS

PostCSS Working with the same principles of preprocessors such as Sass and LESS, it transforms extended syntaxes and features into modern, browser-friendly CSS using JavaScript.

Webpack

Module Bundler Build Tool

“webpack is a module bundler for modern JavaScript applications. When webpack processes your application, it recursively builds a dependency graph that includes every module your application needs, then packages all of those modules into a small number of bundles - often only one - to be loaded by the browser”

Git

Version Control

The most popular version control system on the planet. in Git, every developer's working copy of the code is also a repository that can contain the full history of all changes.

Gulp

Build Tool

Gulp is a build tool that can help you automate a lot of the tasks you need to do as a webdeveloper. Stuff like minifying CSS, JavaScript and images. If you have a task that is repetitive changes are Gulp has a plugin that can do it for you.

Babel

Transpiler

Babel will help turn your modern JavaScript code into older JavaScript. This is great because you can write your code with all the super cool new features and not have to worry about older browsers not supporting it.

iTerm

Terminal/CLI

iTerm is one way to use the Terminal. The Terminal is essential in today's modern web world. iTerm is a free program to make the OS version of Terminal a little more fun to use.

Node

JavaScript Runtime

JavaScript Runtime built on Chrome's V8 JavaScript Engine. Node does a lot of incredible stuff but one thing I use it for is server side JavaScript
