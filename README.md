# My Portfolio
# Installation Instructions
* If on a Mac I recommend installing Homebrew

## Requirements
* Node
    - `$ brew install node`
* Gulp (Install globally)
    - `$ npm install --global gulp-cli`
* Yarn
    - `$ brew install yarn`

### Installation Instructions
* Install all node modules using yarn
    - `$ yarn install`

### Run the app
`$ gulp watch`
